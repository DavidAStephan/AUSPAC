# AUSPAC — A Semi-Structural Macroeconomic Model for Australia

**AU-PAC** is the Australian adaptation of the Banque de France's FR-BDF
model (Lemoine et al., 2019, [WP #736](references/wp736.pdf); updated 2026 in
[WP #1044](references/FR-BDF-update.pdf)). It is a semi-structural
macroeconomic model with Polynomial Adjustment Costs (PAC), explicit
expectations from an E-SAT satellite VAR, and a CES supply block, estimated on
Australian data 1993Q1–2024Q4. The model has **180 declared endogenous
variables** (≈195 after Dynare auxiliary-variable expansion), **53 exogenous
shocks**, and **351 parameters**; five PAC behavioural equations govern
value-added prices, consumption, business investment, household investment, and
employment.

Status: **v4.0 — equation-by-equation OLS production line.** Production
parameters are estimated **block-by-block by OLS / calibration following
FR-BDF wp1044 §2.2** (no joint MCMC). The model is Blanchard-Kahn stable
(`edim=5`, `max|eig|≈1.087`) and reproduces its headline monetary-transmission
IRFs. Full status, the estimated-vs-calibrated scorecard, and open items are in
[`STATUS.md`](STATUS.md); the working paper is in
[`dynare/AUSPAC_WORKING_PAPER.md`](dynare/AUSPAC_WORKING_PAPER.md).

> **Note:** the earlier full-system Bayesian MCMC pipeline (`au_pac_bayesian.mod`
> and the `simulation/` cherrypick tree) was removed in cleanup `7995ce7`. Any
> LMD / Laplace / MHM figure in older docs is historical.

## Quick start (reproduce the headline IRFs)

Prerequisites: MATLAB **R2026a**, Dynare **7.0-arm64**, Python 3.9+. Detailed
instructions in [`RUNNING.md`](RUNNING.md). MATLAB is invoked by full path
(not on `PATH`).

```matlab
% From dynare/ in MATLAB — solves au_pac.mod and regenerates paper artifacts (~1 min).
% au_pac.mod hardcodes all coefficients, so this needs neither dataset.csv nor MATLAB data-prep.
cd dynare; run('regen_all_artifacts.m');
```

To re-derive the parameters from AU data first (then write them into
`au_pac.mod`):

```matlab
% (once) restore the base dataset if absent:  git show 7995ce7^:dataset.csv > dataset.csv
cd data; run('run_all_l2_ols.m');     % refresh the 5 core PAC blocks -> data/pac_blocks/results_*.mat
```

## Reproducing paper results

All paper IRF figures and tables are regenerated by running
[`dynare/regen_all_artifacts.m`](dynare/regen_all_artifacts.m) under MATLAB
R2026a, which addpaths `~/Applications/Dynare/7.0-arm64/matlab`, runs
`dynare au_pac`, and writes the figures/tables into `dynare/paper_artifacts/`.
The PAC-block coefficient tables (§4) come from `data/pac_blocks/results_*.txt`.

> Historical caveat: §6.2 / §6.4 of the working paper still print Bayesian-era
> LMD numbers from the retired `au_pac_bayesian.mod`; those are not reproducible
> from the current tree (see [`STATUS.md`](STATUS.md)).

## Data sources

| Block         | Source                                                                     |
|---------------|---------------------------------------------------------------------------|
| GDP / output  | ABS Cat. 5206.0 — National Income, Expenditure and Product (quarterly)    |
| Capital stock | ABS Cat. 5204.0 Tables 47/48 — capital and depreciation (annual)          |
| Housing prices| ABS Cat. 6416.0 — Residential Property Price Indexes                       |
| Wages         | ABS Cat. 6345.0 — Wage Price Index                                         |
| Hours / employment | ABS Cat. 6202.0 — Labour Force                                       |
| Cash / mortgage / 10Y | RBA F1 / F11 / F2                                                  |
| Commodity prices | RBA I02 / G3 — RBA Index of Commodity Prices                           |
| Exchange rate | RBA F11 TWI; BIS effective exchange rate indices                          |
| US series     | FRED — output gap, CPI, federal funds rate, US 10Y                         |

Aggregation conventions are documented in
[`dynare/au_pac_model_data_README.md`](dynare/au_pac_model_data_README.md).
